1. About CardSync

CardSync is an early-stage software project operated by an independent developer based in the Netherlands. This service is currently offered as part of a closed beta.

CardSync consists of a Chrome browser extension and a web-based dashboard that help Cardmarket sellers export order and shipping data to shipping providers such as MyParcel and PostNL. The service is accessible at cardsync.eu.

Contact: info@cardsync.eu

3. Service Description

CardSync provides the following functionality:

  • A Chrome extension that reads order data from Cardmarket order pages and imports tracking codes from MyParcel and PostNL.
  • A backend that stores synced order data and allows you to export it as CSV files compatible with shipping providers.
  • A dashboard to manage your synced orders, import tracking codes, and export shipping data.

CardSync is a tool that assists with data transfer between platforms. It is not affiliated with, endorsed by, or partnered with Cardmarket, MyParcel, or PostNL.
